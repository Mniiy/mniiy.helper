
# Mniiy.helper

Bot auxiliar simples para o meu servidor :3 (Codado de forma horrivel, uso não recomendado.)
## Instalação

```bash
# Clone github repo
  git clone https://github.com/Mniiy/mniiy.helper.git
  cd ~/mniiy.helper/

# Install requirements (use python3 -m venv .env)
  pip install -r requirements.txt

# (Add your token to .env file)
  echo -e "<token here>" > .env

  python3 start.py
```
